# Kenobi-S1
 DIY mount for astrophotography based on harmonic reducer
 # To the best performance, it is recommended to use the following gearboxes:
# RA axis: LHN-17-100 (low kinematic error)
# DEC axis: HBK-17-100 (low cost)
Parts:
